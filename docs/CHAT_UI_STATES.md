## Keyboard Shortcuts Reference

| Shortcut | Action | Context |
|----------|--------|---------|
| **Space** | Start/Stop recording | When Record button focused |
| **Ctrl+Enter** | Send message | Text input focused OR Ready state |
| **Escape** | Cancel/Close | Any modal or panel open |
| **Ctrl+C** | Copy selected message | Message focused |
| **Delete** | Delete message | User message focused (with confirmation) |
| **Tab** | Navigate elements | Any |
| **Shift+Tab** | Navigate backward | Any |
| **D** | Switch to Data tab | Not in input field |
| **S** | Open Settings | Not in input field |
